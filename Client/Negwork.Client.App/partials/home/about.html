General Info

Negwork is an open-source school project.

It is used to share interesting articles and images with everyone.

Everyone is eligible to register and enjoy the content

Registered users can access details for the articles, create new articles and categories, sort and filter articles, rate articles and comment on articles.

Created by: Veselin Minev

List of tools used

Here is a list of everything I used in the creation of this project.

Name Used for
ASP.NET WebApi Defines endpoints for accessing the database and providing authentication
SQL Server 2014 Storage for the articles, users, images, categories, comments, ratings
EntityFramework Access to the database through the server
Automapper Easier mapping for database models to WebApi request and response models
Ninject Dependency injection in the backend
AngularJS Creating the single-page client application
AngularJS Route Routing for the SPA
AngularJS Cookies Store login information for future visit to the application
AngularJS Animate Animations when switching route
AngularJS Sanitize Visualization of new lines in articles and user profile
Toastr Visualization of errors and success messages
jQuery UI Datepicker for the registration
Bootstrap Styles for the client application
GitHub Storing the source code
Azure Hosting the server in the cloud
Heroku Hosting the client application in the cloud
